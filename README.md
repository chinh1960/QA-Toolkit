🛠️ QA Toolkit v10.4.87
QA Toolkit là một công cụ tối ưu hóa hệ thống Windows mạnh mẽ, hiện đã được nâng cấp toàn diện từ nền tảng PowerShell sang Python. Công cụ giúp tăng tốc máy tính, dọn dẹp tệp rác và giám sát hiệu suất hệ thống với giao diện hiện đại.

🚀 Tính năng nổi bật
Core Python Engine: Xử lý logic hệ thống nhanh hơn và ổn định hơn so với script truyền thống.

Giao diện CustomTkinter: Giao diện người dùng (GUI) hiện đại, hỗ trợ đầy đủ chế độ Dark Mode và Light Mode.

Multi-threading: Các tác vụ dọn dẹp và kiểm tra chạy dưới dạng luồng riêng biệt, không gây hiện tượng treo ứng dụng (Not Responding).

Sửa lỗi Task 5 & 8: Khắc phục triệt để lỗi crash khi dọn dẹp các tệp tạm thời hệ thống.

Báo cáo hệ thống: Xuất báo cáo tình trạng máy tính dưới định dạng chuyên nghiệp.
