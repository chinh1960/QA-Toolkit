QA Toolkit Documentation

Guide for using QA Toolkit

System Audit

System Audit analyzes your computer hardware and operating system. It checks CPU, RAM, disk usage and Windows configuration.

Temp Cleanup

Temp Cleanup removes unnecessary files including:

  • Temporary files
  • Windows cache
  • Update leftovers

Startup Manager

Allows users to view and disable programs that start with Windows.

System Health Score

The system health score ranges from 0 to 100 and evaluates the system condition.

HTML Report

QA Toolkit can export system information into a professional HTML report.
