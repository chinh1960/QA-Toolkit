✅ Cách tốt nhất (khuyến nghị):
Tải QA-Toolkit.zip thay vì .exe
  • Giải nén file zip
  • Chuột phải vào QA-Toolkit.exe → Properties → tick “Unblock” → OK
  • Double-click để chạy (hoặc chuột phải → Run as administrator)
Hoặc tải trực tiếp .exe:
Nhấn “More info” → “Run anyway” khi Windows chặn. File .zip ít bị chặn hơn rất nhiều. Bạn chỉ cần làm 1 lần là xong.
